HoopFire

HoopFire is a sports prediction web application. Two friends and I came up with the idea a few years back and it has been a pet project ever since. It was originally built using Vaadin and was rewritten in 2016. The front end is now built using AngularJS with Angular Material design principles. The data is persisted using Hibernate against a MySQL database with application dependencies being managed by Spring. The application build is handled by Maven and we're using Heroku to host the application.

WhereWeather

WhereWeather is a very simple and straightforward weather forecast website. It asks for the users location, but can also get the weather of any user inputted location. The forecast goes forward a few days and displays the information based on the forecast, as projected by the Dark Sky API. The site was built using Yeoman to set up the scaffolding, which includes NPM, Bower, and Grunt. The site is implemented with AngularJS and Angular Material Design. All icons and logos I created myself with Adobe Illustrator.

Plex Request

Plex Request started as a joke between my friends and I. We wanted a way to browse movies and Tv shows we may want to watch later. It simply sends a user selected list to a friend of mine who gracioulsy volunteered to maintain the full list. All information is a product of the JSON returned from using the The MovieDB API. The site was built using Yeoman to set up the scaffolding, which includes NPM, Bower, and Grunt. The site is implemented with AngularJS and Angular Material Design.
