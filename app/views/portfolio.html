HoopFire

HoopFire is a sports prediction web application. Two friends and I came up with the idea a few years back and it has been a pet project ever since. It was originally built using Vaadin and was rewritten in 2016. The front end is now built using AngularJS with Angular Material design principles. The data is persisted using Hibernate against a MySQL database with application dependencies being managed by Spring. The application build is handled by Maven. Unfortunately, the application is currently not hosted anywhere, but the code can be found here.
